# Dispoll

Dispoll allows you to send poll in a **simple command** in a Discord channel.

![Dispoll Gif](https://github.com/HZooly/Dispoll/blob/master/.github/dispoll.gif)

## Installation

### Setup
```
npm install
```

### Run development
```
npm run dev
```

### Build
```
npm run build
```

### Run build
```
npm start
```
